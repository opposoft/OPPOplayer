## 🏗️ О проекте

**ОППО** — это проект для тех, кто хочет смотреть новости не из одного источника, а сразу из множества.

Здесь собраны телеканалы и радиостанции из разных стран, чтобы можно было:
- сравнивать подачу;
- видеть разные взгляды;
- формировать более объёмную картину мира.

Сайт сделан максимально просто:
**без шума, без перегруза, без лишнего интерфейса**.

---

## ⚡ Особенности

- 📺 ТВ-каналы и 📻 радиостанции в одном месте
- 🔍 поиск по списку каналов
- ⭐ избранное
- 🎚 выбор качества потока
- 💾 сохранение настроек в `localStorage`
- 🌐 поддержка HLS через `hls.js`
- ▶️ поддержка YouTube-стримов
- 🎧 визуализатор для радио
- 📱 адаптивный интерфейс для мобильных устройств

---

## 🧠 Идея

Этот проект создан как **альтернатива информационному пузырю**.

Цель — дать доступ к разным источникам, чтобы пользователь сам мог:
- сопоставлять новости;
- замечать различия в подаче;
- делать собственные выводы.

**ОППО** не навязывает мнение. **ОППО** показывает источники!

---

## 🌍 Источники

Все потоки берутся с **официальных сайтов**.

Проект:
- не хранит видео у себя;
- не ретранслирует контент;
- не требует подписки;
- не использует региональные ограничения там, где поток открыт официально.
